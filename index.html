Construyendo un landing page

Se realiza un landing page implementando CSS, utilizando archivos de hojas externas al documento HTML y de base para homogeneizar el diseño del sitio. Cuenta con una sección principal, de productos y footer.

Se realiza un landing page a partir de una maqueta, siguiendo las especificaciones de la misma. El proyecto cuenta con un arbol de archivos ordenados, se utiliza FLEX, y se implementan íconos con Font Awesome.
